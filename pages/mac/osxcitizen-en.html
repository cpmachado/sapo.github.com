OSX Citizen identify yourself

What is it?

This is an implementation of Portuguese Citizen Card support atop Opensc.Tokend, with native-grade Mac OS X integration.

Why?

The government-issued Portuguese Citizen Card software for Mac was, unfortunately, only compatible with Firefox.

Plus its sub-par integration meant that it did not take advantage of Mac OS X native security frameworks, therefore lacking proper keychain integration (and, in consequence, lack of support for Mail.app and Safari).

This project fixes that by reimplementing the required functionality atop OpenSC Tokend, a standards-compliant, cross-platform framework that allows for fully native integration and extensibility, thereby vastly improving the user experience and allowing for further app integration.

How can I help?

Test the binary packages and report any issues that you may come across.

How do I get started?

Download the latest binary bundle and run the installer:

  • For Snow Leopard (10.6) and Lion (10.7)
  • For Tiger (10.4) and Leopard (10.5)

You can also the latest version of the source code from our Git repo. For checking out the code use the git command line client as follows:

git clone git://github.com/sapo/osxcitizen.git
