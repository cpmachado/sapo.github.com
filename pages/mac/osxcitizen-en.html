OSX Citizen identify yourself

What is it?

This was the first fully working implementation of Portuguese Citizen Card support atop Opensc.Tokend, with native-grade Mac OS X integration.

Why?

The government-issued Portuguese Citizen Card software for Mac was, unfortunately, only compatible with Firefox.

That has since been fixed (there is now a fully working set of official packages), but the original version did not take advantage of Mac OS X native security frameworks and lacked proper keychain integration (and, in consequence, lack of support for Mail.app and Safari).

This project fixed that by reimplementing the required functionality atop OpenSC Tokend, a standards-compliant, cross-platform framework that allowed for fully native integration and extensibility, thereby vastly improving the user experience and allowing for further app integration.
