OSX Citizen identifique-se

O que é?

É uma implementação do cartão do cidadão sobre Opensc.Tokend bem integrada com o OSX.

Porquê?

A introdução do cartão do cidadão em Portugal foi acompanhada do lançamento oficial de pacotes de software para os sistemas operativos Windows, OSX e Linux. Infelizmente, a versão Mac lançada, é apenas compatível com o browser Firefox.

A implementação escolhida não nos parece a melhor pois não aproveita os frameworks nativos do OSX. O resultado é uma inexistência de integração com o Keychain e consequentemente com o Safari e Mail.app. Este projecto pretende resolver o problema com uma implementação sobre o OpenSC Tokend, uma framework disponível em várias plataformas e assim potenciar o aparecimento de novas aplicações que façam uso das potencialidades do cartão do cidadão.

Como posso contribuir?

Experimentando o package distribuído e reportando eventuais bugs/erros.

Como posso usar?

Descarregue o instalador e execute-o:

  • Para Snow Leopard (10.6) e Lion (10.7)
  • Para Tiger (10.4) e Leopard (10.5)

Pode também obter a última versão do código fonte do nosso repositório, com a seguinte linha de comando:

git clone git://github.com/sapo/osxcitizen.git
