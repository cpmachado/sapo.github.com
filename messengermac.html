SAPO Messenger for Mac

What is it?

SAPO Messenger for Mac is a free Jabber/XMPP client for the Mac OS X platform, developed at SAPO and released under a GNU GPL2 license.

Features

Major features include:

  • XMPP-compliant client, based on the robust Psi core;
  • clean Mac OS X experience;
  • multi-contact support;
  • WebKit-based chat windows;
  • Standards-based file transfer;
  • Uniform roster across multiple accounts, with filtering search of contacts;
  • Message center to manage offline messages and headline-type messages;
  • Support for MSN transport on any account;
  • SMS integration (requires SAPO account);
  • Growl support;
  • Sparkle-based updates.

Get the source

You clone the source here at github from our official repository. 

git clone git@github.com:sapo/sapo-messenger-for-mac.git

Binaries

You can follow the latest releases in two ways:

Download the latest nightly build of SAPO Messenger Mac. It uses the Sparkle framework to keep you up-to-date;

subscribe to the nightly builds feed or select "Nightly Builds" in the "Auto-Update" menu that is available in the "General" pane of the application Preferences window.

The current builds have been stable for day-to-day use for some months now.
